# Kusunoki Dictioanry Management Tools
Using Python 3

## Dependencies
### Required
- Python >= 3.6
- Python Packages
  - ruamel.yaml
  - click
  - lark-parser
  - pandas
  - dataclasses

### Recommended
- Python >= 3.7
- pip

## Installation
### If you want to use it as a python package
```sh
pip install (--user) -e .
```

### If you want to use it via commandline
Either do `pip install` or add `Kusunoki/bin` to `${PATH}` of your shell.

## Uninstallation
```sh
pip remove pyksnk
```
